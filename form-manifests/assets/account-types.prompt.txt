Nano Banana (Gemini 2.5 Flash Image) prompt — account-type icon sheet
=====================================================================

Use the prompt below to generate ONE composite PNG, then drop it at
  public/img/account-types.png
The add/edit-account picker (src/components/accounts/AccountForm.tsx) slices it
via the grid declared in src/lib/accountTypes.ts
(SPRITE_COLS = 4, SPRITE_ROWS = 5, 256 px tiles, gutter = 0). Until the file is
present the picker renders inline lucide icons with labels — no crash.

Target output: 1024 × 1280 PNG, sRGB, no transparency.

    warm gold #E0B96A · soft rose #D49AA0

  across all icons — none should dominate
